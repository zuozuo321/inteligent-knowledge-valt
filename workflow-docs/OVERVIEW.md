# 知识库工作流系统总览

## 系统架构

```
+-----------+     +-------------+     +------------------+     +------------+
|  用户     | --> | VSCode      | --> | Claude Code      | --> | 知识库     |
| (提问/    |     | (集成开发   |     | (AI 引擎)        |     | (Obsidian  |
|  整理)    |     |  环境)      |     |                  |     |  Vault)    |
+-----------+     +-------------+     +------------------+     +------------+
                        |                      |                     |
                        |                      v                     |
                        |              +------------------+          |
                        |              | MCP 服务端       |          |
                        +------------->|                  |<---------+
                                       | - obsidian-mcp   |
                                       | - bilibili-mcp   |
                                       +------------------+
                                                |
                                                v
                                       +------------------+
                                       | 外部服务        |
                                       | - B站 API       |
                                       | - Whisper (本地)|
                                       +------------------+
```

## 六大组件

### 1. ccswitch (API 代理)
- 位置：本地代理服务
- 功能：管理 API 密钥、请求路由、模型映射
- 作用：将 Claude Code 的 API 请求转发到兼容的模型后端

### 2. VSCode (集成开发环境)
- 位置：开发者桌面
- 功能：提供编辑器界面，集成 Claude Code 扩展
- 作用：AI 交互的主要界面，文件编辑、代码执行

### 3. Claude Code (AI 引擎)
- 位置：VSCode 扩展 / 命令行
- 功能：接收用户指令，调用 MCP 工具，操作知识库
- 作用：工作流的智能核心，负责理解、推理、执行

### 4. Obsidian (可视化知识库)
- 位置：用户桌面
- 功能：Markdown 笔记管理，双向链接，图谱视图
- 作用：知识的可视化展示和手动编辑

### 5. Claudian (会话管理)
- 位置：VSCode 扩展
- 功能：管理 AI 会话上下文、历史记录
- 作用：保持会话连续性，避免上下文丢失

### 6. 知识库 (Vault)
- 位置：`$VAULT_PATH/`
- 功能：结构化存储所有知识，按 Karpathy 三层架构组织
- 作用：知识的持久化存储和检索

## 数据流

```
                    +------------------+
                    |  素材获取        |
                    |  (B站/网页/笔记) |
                    +--------+---------+
                             |
                             v
                    +------------------+
                    |  raw/ (原始层)   |
                    +--------+---------+
                             |
                    AI 编译 (Compile)
                             |
                             v
                    +------------------+
                    |  wiki/ (知识层)  |
                    +--------+---------+
                             |
                    AI 查询 (Query)
                             |
                             v
                    +------------------+
                    |  合成回答给用户  |
                    +--------+---------+
                             |
                    有价值内容回填
                    (Backfill)
                             |
                             v
                    +------------------+
                    | wiki/ 或 outputs/|
                    +------------------+
```

## 快速开始

1. **部署环境**：运行 `deploy/deploy-all.ps1` 一键部署
2. **配置知识库**：编辑 `config/claude-settings.json`，替换占位符
3. **启动 Obsidian**：打开 `$VAULT_PATH`
4. **启动 VSCode**：在项目根目录中打开
5. **开始使用**：在 Claude Code 中提问或导入素材
